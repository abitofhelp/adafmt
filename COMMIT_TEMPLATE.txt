feat(project)!: initial public release

Establishes the stable 1.x CLI and formatter behavior for adafmt.
- Ada Language Server integration via JSON-RPC with LSP TextEdit processing
- Multiple UI modes (pretty/basic/plain/off) with automatic terminal capability detection
- Robust error handling with retry logic and GNATFORMAT false positive detection
- Atomic file operations with dry-run by default and explicit write mode
- GPR project integration with Alire workspace detection and path filtering
- ALS process lifecycle management with preflight conflict resolution
- JSONL structured logging and comprehensive debugging capabilities
- CI-friendly check mode with unified diff preview

BREAKING CHANGE: First stable release; prior pre-1.0 behavior is not guaranteed compatible.
